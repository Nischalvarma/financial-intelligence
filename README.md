# Quant Trading Dashboard

Machine Learning-Based Financial Analytics and Backtesting System built using Python, Streamlit, and Quantitative Trading Concepts.

## Features

- Financial data ingestion pipeline
- Technical indicator engineering
- Machine learning-based signal generation
- Backtesting framework
- Interactive dashboard using Streamlit
- Trading signal visualization
- RSI and trend analysis

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Streamlit
- Plotly
- yFinance
